# 프로젝트 구조 - 가이드

이 문서는 Picake 프로젝트의 전체 구조를 설명합니다.

## 📁 프로젝트 구조

```
picake/
├─ package.json             # 루트 설정
├─ .yarnrc.yml              # Yarn Berry 설정
├─ node_modules/            # 의존성 패키지들 (node_modules 방식)
├─ tsconfig.base.json       # TypeScript 기본 설정
├─ tsconfig.json            # TypeScript 설정
├─ eslint.config.js         # ESLint 설정
├─ vercel.json              # Vercel 배포 설정
├─ yarn.lock                # Yarn 의존성 잠금 파일
├─ apps/                    # 애플리케이션들
│  ├─ backend/              # 백엔드 서비스 (NestJS + TypeScript)
│  │  ├─ src/               # 소스 코드
│  │  │  ├─ apis/           # API 모듈들 (Consumer, Seller, Admin)
│  │  │  │  ├─ consumer/    # 구매자 API
│  │  │  │  ├─ seller/      # 판매자 API
│  │  │  │  └─ admin/       # 관리자 API
│  │  │  ├─ modules/        # 비즈니스 로직 모듈들
│  │  │  │  ├─ auth/        # 인증 모듈
│  │  │  │  ├─ product/     # 상품 모듈
│  │  │  │  ├─ store/       # 스토어 모듈
│  │  │  │  ├─ order/       # 주문 모듈
│  │  │  │  ├─ feed/        # 피드 모듈
│  │  │  │  ├─ review/      # 리뷰 모듈
│  │  │  │  ├─ like/        # 좋아요 모듈
│  │  │  │  ├─ recent-view/ # 최근 본 상품 모듈
│  │  │  │  ├─ notification/ # 알림 모듈
│  │  │  │  ├─ fcm/         # FCM 푸시 알림 모듈
│  │  │  │  ├─ chat/        # 채팅 모듈 (Socket.IO)
│  │  │  │  ├─ statistics/  # 통계 모듈
│  │  │  │  ├─ business/    # 사업자 인증 모듈
│  │  │  │  ├─ seller-home/ # 판매자 홈 모듈
│  │  │  │  └─ upload/      # 파일 업로드 모듈
│  │  │  ├─ common/         # 공통 유틸리티
│  │  │  └─ infra/          # 인프라 설정 (데이터베이스, Prisma)
│  │  ├─ dist/              # 빌드 산출물
│  │  ├─ nest-cli.json      # Nest CLI 설정
│  │  └─ package.json       # 백엔드 의존성/스크립트
│  ├─ web-user/             # 사용자 웹 애플리케이션 (Next.js + TypeScript)
│  │  ├─ src/               # 소스 코드
│  │  │  ├─ app/            # Next.js App Router
│  │  │  │  ├─ layout.tsx   # 루트 레이아웃
│  │  │  │  ├─ page.tsx     # 홈페이지
│  │  │  │  ├─ auth/        # 인증 페이지 (login, register, find-account, reset-password)
│  │  │  │  ├─ product/     # 상품 상세 페이지
│  │  │  │  ├─ store/       # 스토어 상세 페이지
│  │  │  │  ├─ order/       # 주문 관련 페이지
│  │  │  │  ├─ reservation/ # 예약 페이지
│  │  │  │  ├─ mypage/      # 마이페이지
│  │  │  │  ├─ saved/       # 저장/좋아요 목록
│  │  │  │  ├─ search/      # 검색 페이지
│  │  │  │  ├─ map/         # 지도 페이지
│  │  │  │  ├─ chat/        # 채팅 페이지
│  │  │  │  ├─ alarm/       # 알림 페이지
│  │  │  │  └─ qa/          # Q&A 페이지
│  │  │  ├─ common/         # 공통 컴포넌트 및 유틸리티
│  │  │  │  ├─ components/  # 재사용 가능한 컴포넌트
│  │  │  │  ├─ config/      # 설정 파일
│  │  │  │  ├─ constants/   # 상수
│  │  │  │  ├─ store/       # 전역 상태 (Zustand)
│  │  │  │  ├─ types/       # 타입 정의
│  │  │  │  └─ utils/       # 유틸리티 함수
│  │  │  ├─ features/       # 기능별 모듈
│  │  │  │  ├─ auth/        # 인증
│  │  │  │  ├─ product/     # 상품
│  │  │  │  ├─ store/       # 스토어
│  │  │  │  ├─ order/       # 주문
│  │  │  │  ├─ feed/        # 피드
│  │  │  │  ├─ review/      # 리뷰
│  │  │  │  ├─ like/        # 좋아요
│  │  │  │  ├─ chat/        # 채팅
│  │  │  │  ├─ alarm/       # 알림
│  │  │  │  ├─ mypage/      # 마이페이지
│  │  │  │  ├─ search/      # 검색
│  │  │  │  ├─ upload/      # 업로드
│  │  │  │  ├─ fcm/         # FCM 토큰
│  │  │  │  ├─ recent/      # 최근 본 상품
│  │  │  │  └─ legal/       # 법적 문서
│  │  │  └─ types/          # TypeScript 타입 정의
│  │  ├─ public/            # 정적 파일
│  │  ├─ next.config.ts     # Next.js 설정
│  │  └─ package.json       # 의존성/스크립트
│  ├─ web-seller/           # 판매자 웹 애플리케이션 (React + Vite + Shadcn/ui)
│  │  ├─ src/               # 소스 코드
│  │  │  ├─ common/         # 공통 컴포넌트 및 유틸리티
│  │  │  │  ├─ components/  # 재사용 가능한 컴포넌트 (Shadcn/ui 기반)
│  │  │  │  │  ├─ alerts/   # 알림 컴포넌트
│  │  │  │  │  ├─ fallbacks/ # 로딩/에러 폴백 컴포넌트
│  │  │  │  │  ├─ layouts/  # AdminLayout 등 레이아웃 컴포넌트
│  │  │  │  │  ├─ providers/ # 프로바이더 컴포넌트
│  │  │  │  │  └─ ui/       # Shadcn/ui 컴포넌트 (button, input, card, badge 등)
│  │  │  │  ├─ config/      # 설정 파일
│  │  │  │  ├─ constants/   # 상수 (paths, routes)
│  │  │  │  ├─ hooks/       # 공통 커스텀 훅
│  │  │  │  ├─ store/       # 전역 상태 (Zustand)
│  │  │  │  ├─ styles/      # 전역 스타일
│  │  │  │  ├─ types/       # 타입 정의
│  │  │  │  └─ utils/       # 유틸리티 함수
│  │  │  ├─ features/       # 기능별 모듈
│  │  │  │  ├─ auth/        # 인증
│  │  │  │  ├─ business/    # 사업자 인증
│  │  │  │  ├─ store/       # 스토어 관리
│  │  │  │  ├─ product/     # 상품 관리
│  │  │  │  ├─ order/       # 주문 관리
│  │  │  │  ├─ feed/        # 피드 관리
│  │  │  │  ├─ statistics/  # 통계
│  │  │  │  ├─ notification/ # 알림
│  │  │  │  ├─ chat/        # 채팅
│  │  │  │  ├─ home/        # 판매자 홈
│  │  │  │  ├─ mypage/      # 마이페이지
│  │  │  │  ├─ upload/      # 업로드
│  │  │  │  └─ legal/       # 법적 문서
│  │  │  └─ pages/          # 페이지 컴포넌트
│  │  ├─ index.html         # HTML 엔트리 포인트
│  │  ├─ vite.config.ts     # Vite 설정
│  │  └─ package.json       # 의존성/스크립트
│  └─ web-admin/            # 관리자 웹 애플리케이션 (React + Vite + Shadcn/ui)
│     ├─ src/               # 소스 코드
│     │  ├─ common/         # 공통 컴포넌트 및 유틸리티
│     │  │  ├─ components/  # 재사용 가능한 컴포넌트
│     │  │  ├─ config/      # 설정 파일
│     │  │  ├─ constants/   # 상수 (paths, routes)
│     │  │  ├─ store/       # 전역 상태 (Zustand)
│     │  │  ├─ styles/      # 전역 스타일
│     │  │  ├─ types/       # 타입 정의
│     │  │  └─ utils/       # 유틸리티 함수
│     │  ├─ features/       # 기능별 모듈
│     │  │  └─ auth/        # 인증 (TOTP 기반)
│     │  └─ pages/          # 페이지 컴포넌트
│     │     ├─ Root.tsx     # 관리자 홈
│     │     └─ auth/        # 로그인, 회원가입, TOTP 설정/검증
│     ├─ index.html         # HTML 엔트리 포인트
│     ├─ vite.config.ts     # Vite 설정
│     └─ package.json       # 의존성/스크립트
├─ docs/                    # 프로젝트 문서
│  ├─ backend/              # 백엔드 관련 문서
│  ├─ common/               # 공통 문서
│  ├─ web-user/             # 사용자 웹 관련 문서
│  ├─ web-seller/           # 판매자 웹 관련 문서
│  └─ infra/                # 인프라 관련 문서
│     ├─ aws/               # AWS 인프라 가이드
│     └─ vercel/            # Vercel 배포 가이드
└─ packages/                # 공유 패키지들 (향후 구현 예정)
```
