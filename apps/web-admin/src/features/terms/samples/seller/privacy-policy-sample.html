개인정보 처리방침

시행일: 2026년   6월   30일

두루랩스(픽 케이크, Picake)는 이용자의 개인정보를 중요하게 생각하며, 「개인정보 보호법」 등 관련 법령을 준수합니다.
본 방침을 통해 개인정보가 어떻게 수집·이용·보호되는지 안내드립니다.

제1조 (수집하는 개인정보 항목 및 수집 방법)

① 일반회원(구매자) 가입 시

수집 항목 수집 방법 필수
소셜 서비스 식별값(Google ID 또는 Kakao ID) 소셜 로그인 연동 필수
소셜 계정 이메일 주소 소셜 로그인 연동 필수
이름(성명) 가입 화면 직접 입력 필수
휴대폰 번호 직접 입력 + SMS 인증 필수

* 소셜 로그인 연동 시 닉네임·프로필 사진은 수집하지 않습니다.

② 주문 및 픽업 예약 시

수집 항목 수집 방법 필수
예약자 이름 주문 화면 직접 입력 선택
예약자 휴대폰 번호 주문 화면 직접 입력 선택
레터링 문구 (최대 500자) 주문 화면 직접 입력 선택
케이크 디자인 요청 이미지 파일 업로드 선택
입금자 이름 입금 확인 화면 직접 입력 필수(입금대기 시)
환불 계좌 (은행명, 계좌번호, 예금주) 환불 신청 화면 직접 입력 해당 시 필수

③ 서비스 이용 중 자동 수집

수집 항목 수집 방법
접속 IP 주소 서버 로그 자동 기록
기기 정보(기기 ID, OS 버전 등) 앱·브라우저 자동 수집
서비스 이용 기록 서버 로그 자동 기록
FCM 디바이스 토큰 앱 알림 권한 허용 시 자동 수집(앱 한정)

제2조 (개인정보의 수집 및 이용 목적)

목적 처리 항목
회원 식별, 소셜 로그인 처리 및 계정 관리 소셜 ID, 이메일, 이름, 휴대폰 번호
휴대폰 본인 인증 휴대폰 번호, 인증번호
케이크 주문·예약·픽업 서비스 제공 주문 정보, 예약자 정보, 레터링·이미지
주문 정보 판매회원 전달 예약자명, 연락처, 주문 상세 내역
앱 푸시 알림 발송 FCM 디바이스 토큰
고객 문의·민원·분쟁 처리 이름, 휴대폰 번호, 주문 내역
서비스 개선, 통계 분석, 부정이용 탐지 서비스 이용 기록, 접속 IP

제3조 (개인정보의 제3자 제공)

회사는 원활한 주문 중개를 위해 아래와 같이 개인정보를 제3자에게 제공합니다.
회원가입 시 이에 대한 별도 동의를 받습니다.
항목 내용
제공받는 자 구매자가 주문한 해당 판매회원(스토어)
제공 목적 케이크 제작, 주문 내용 확인, 픽업 시 예약자 본인 확인, 문의·상담
제공 항목 예약자 이름, 휴대폰 번호, 레터링 문구, 디자인 요청 이미지, 주문 상세 내역(사이즈·맛·수량·픽업 일시 등)
보유 및 이용 기간 주문 서비스 제공 완료 및 분쟁 해결 시까지. 이후 즉시 파기. 단, 법령에 의한 보존 기간은 제4조에 따름.

이 외의 제3자 제공은 이용자의 사전 동의 또는 법령 규정에 따른 경우에만 이루어집니다.

제4조 (개인정보의 보유 및 이용 기간)

  1. 회원 정보는 회원 탈퇴 시까지 보유합니다.
  2. 관련 법령에 따라 아래 기간 동안 보관합니다.
    보존 항목 근거 법령 기간
    계약·청약철회에 관한 기록 전자상거래법 5년
    대금결제·재화 공급에 관한 기록 전자상거래법 5년
    소비자 불만·분쟁처리 기록 전자상거래법 3년
    전기통신일지(접속 로그, IP) 통신비밀보호법 3개월
    위치정보 이용·제공사실 확인자료 위치정보법 6개월
  3. 탈퇴 후에는 개인정보를 즉시 파기하되, 위 법정 보존 기간이 남아 있는 경우 분리 보관합니다.
  4. 12개월 이상 로그인하지 않은 장기 미이용 회원의 경우, 별도 분리 보관 또는 파기 조치를 취합니다.

제7조 (개인정보의 파기)

  1. 보유 기간 경과 또는 처리 목적 달성 시 지체 없이 파기합니다.
  2. 전자적 파일: 복구 불가능한 방법으로 영구 삭제
  3. 출력물·종이 문서: 분쇄 또는 소각

제8조 (이용자의 권리 및 행사 방법)

  1. 열람·수정: 서비스 내 마이페이지에서 본인 정보를 직접 확인하고 수정할 수 있습니다.
  2. 삭제(탈퇴): 마이페이지 → 내 계정 정보 → 회원탈퇴를 통해 요청할 수 있습니다.
  3. 동의 철회: 마케팅·알림 동의는 설정에서 언제든지 철회할 수 있습니다.
  4. 픽업자 정보(본인이 아닌 제3자)를 입력하는 경우, 해당 정보 제공에 관한 사전 동의를 취득한 것으로 간주합니다.

제9조 (개인정보의 안전성 확보 조치)

  1. 접근 제어: 개인정보 처리 시스템 접근 권한을 최소 인원으로 제한
  2. 전송 암호화: HTTPS/TLS를 통한 암호화 전송
  3. 저장 보호: AWS S3 서버 측 암호화(AES-256) 적용
  4. 인증 토큰 관리: JWT 기반 인증으로 토큰 유효기간 관리
  5. 임직원 교육: 개인정보 취급 담당자 최소화 및 정기 교육 실시

제10조 (쿠키 등)

서비스는 로그인 유지를 위해 JWT 방식을 사용하며, 해당 토큰은 브라우저 localStorage에 저장됩니다. 서비스 편의 및 분석을 위해 쿠키를 사용할 수 있으며, 브라우저 설정을 통해 쿠키 저장을 거부할 수 있습니다.
